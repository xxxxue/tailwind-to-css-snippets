# tailwind-to-css-snippets

Use Tailwind CSS utility classes to write CSS, LESS, and SCSS code.

`flex` `w-full` `w-screen` and more

## Install

- Search for `xxxxue.tailwindcss-to-css-snippets` in VS Code
- [Marketplace](https://marketplace.visualstudio.com/items?itemName=xxxxue.tailwindcss-to-css-snippets)
